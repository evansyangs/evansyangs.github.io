路由器固件更新与桥接设置

提示

今天我又断网了吗...

最近被断断续续的无线网逼疯，翻箱倒柜找出了一个TP-Link WR842N，想到可以无线桥接组网，解决路由器位置放置不合理的问题，增强一下无线信号强度。

# 主副路由器

主路由器：TL-WR842N 4.0 (opens new window)

副路由器：TL-WR886N 6.0 (opens new window)

两个巨坑路由器（大众产品，销量好，价格合适，缩水...）

# 固件更新

# TL-WR842N 4.0

当前软件版本 20140120

官方教程：TL-WR842N 软件升级方法 (opens new window)

官方固件：TL-WR842N V4.0_140528标准版 (opens new window)

固件对比：

版本 特点 推荐星级
20140528 (opens new window) 官网最新，旧版界面 ★★
20140929 (opens new window) 可能是842N+的固件，未确认
20150130 (opens new window) 很稳定，旧版界面，测试版，工程师提供 ★★★
20141118 (opens new window) 比20150425稳定些，新版界面，工程师提供 ★★★★
20150425 (opens new window) 不稳定，新版界面 ★★

我现在用的是2014118版本固件，WDS的中继效果稳定和速度有较大幅度提升，新版有访客网络功能，比较实用。

提示

若需要从20150425版降级，请先将路由器恢复出厂设置，再刷到20150130版（20141118也是新版界面，比150425稳定）

TP-Link的工程师建议

842N v4网段不要用0网段，用2-9之间，wan拨号的话，用正常模式，不用自动，lan口ip也选手动，不要自动（理论上意义不大，但确实稳定）wan口速率非必要，无需更改，mac地址无需克隆(非固定接入线路强制绑定，无需克隆)

来自 Anywlan论坛 (opens new window)

# TL-WR886N 6.0

官方教程：TL-WR886N 4.0及以上的版本均云路由器，该路由器支持在线升级，无需手动下载升级程序 (opens new window)

第三方固件：动手能力比较强的可以尝试刷SuperWRT (opens new window)
