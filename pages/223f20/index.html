苏州美食

提示

苏州吃吃喝喝！

# 路线推荐

三元坊地铁站：书院巷 <- 十全街 -> 葑门横街

临顿路地铁站：定慧寺巷 -> 双塔市集

察院场地铁站：观前街 -> 皮市街 -> 西北街

山塘街地铁站：山塘街 -> 西中市 -> 吴趋坊

# 分类推荐

标记：老字号[1]

早餐：

  • 大饼：张记大饼店、余暨大饼店、王氏林记烧饼、潘记隆烧饼

  • 汤圆：老街汤圆、朱新年点心店

  • 粽子：周妈妈香粽店、陈记一品斋粽子

  • 蛋饼：喜有福、老王蛋饼、姑苏区老黄苏式蛋饼、宗霞鸡蛋饼

  • 馄饨：老意和、矮脚楼馄饨、乐惠馄饨、陈记泡泡馄饨

  • 生煎：裕永盛、哑巴生煎

  • 锅贴：小姜锅贴、老城隍庙锅贴

  • 杂粮煎饼：小热荤

  • 月饼：长发月饼

  • 包子：蛮好阁包子馆、义昌福[1:1]

  • 汤包：山塘老街、吴盛兴、三德利绉纱汤包馆、老陆家汤包馆

  • 豆浆：西北街豆浆店、闾门姚记豆浆

  • 鼎盛鲜，马栋佩烧卖，江南蟹壳皇

苏帮菜：

  • 五卅酒家、老苏州茶酒楼、半月斋、吴记小园楼、松鹤楼[1:2]

面食：

  • 同得兴、万泰兴、面见阳春、松鹤面馆、近水台、聚新春、天桥炒面、汤香源面馆、美味斋、原怡园面馆、朱鸿兴

甜食：

  • 鸡脚：黎里高记辣脚、祥鑫饮食店、黎里王记辣脚

  • 绿豆汤：紫笑香、赤小豆饮食店

  • 赤豆元宵：大孃孃、潘玉麟糖粥

  • 豆花：木缘新豆花

糕点：

  • 黄富兴糕团、沈记青团、近水楼、棠芝坊、明月楼糕团、采芝斋[1:3]、叶受和[1:4]、稻花香[1:5]、黄天源[1:6]、杨招娣糕点、万福兴[1:7]

炸物：

  • 申城特色爆鱼、赵天禄[1:8]、馋佬沛、荣阳楼

其它：

  • 兴奋羊肉老店、陆家市食品店、杜三珍卤菜[1:9]、陆稿荐[1:10]、十年麻辣烫、剑雄炒货蜜饯店、老陆稿荐[1:11]、五德居、东方卤菜店、阿二烤鸡

# 葑门横街

最有烟火气的苏州老街(老菜场)，与十全街东侧相连，苏州大学东校区南边

# 张记大饼店

地址： 横街111号

大饼(芝麻+葱油)、市面少见的点心“老虎脚爪”

# 余暨大饼店

老夫妻经营20多年，价格实在用料足、味道香

地址： 横街86号

咸味大饼(香葱+油渣)、甜味大饼(糖馅)

# 黄富兴糕团

开店20多年

地址： 横街97号

猪油糕、条头糕、赤豆糕、核桃夹糕、薄荷方糕、玫瑰松糕

# 沈记青团

年初三至清明节后两周左右

地址： 横街91号

青团

# 申城特色爆鱼

地址： 横街29号

爆鱼

# 赵天禄

百年老店、苏州老字号

地址： 横街160号

油氽tǔn团子

# 老街汤圆

地址： 横街160号

汤圆馅料丰富、老店

# 周妈妈香粽店

地址： 横街105号

灰汤粽

# 十全街

十全十美，十分好吃

# 五卅酒家

地址： 十全街418号

招牌水煮腰花、糟(zāo)溜鱼片、牛肉煲、椒盐排条

# 老苏州茶酒楼

老牌苏帮菜馆

地址： 十全街658号

豆腐花、手剥虾仁、桂花糖藕、卤鸭

# 同得兴

地址： 十全街624号

枫镇大肉面(5到10月供应、舌尖上的中国)

# 万泰兴

地址： 十全街670号

三虾面、开洋拌面、枫镇大肉面

# 半月斋

店主汪成是苏帮菜制作技艺第三代五位传承大师之一

地址： 十全街916号

时令点心、苏式面、精致小菜：火腿胡萝卜丝酥饼、白斩鸡

# 黎里高记辣脚

地址： 十全街773号

黎里辣脚

# 馋佬沛

地址： 十全街776号

面拖排骨、芝士鸡腿、芝士萝卜丝饼

# 祥鑫饮食店

地址： 十全街783号

鸡脚、绿豆汤、咖喱鸡肉卷

# 近水楼

苏州老牌糕团店

地址： 十全街138号(分店：吴趋坊96号、皮市街268号)

炒肉馅团子(夏季)、猪油糕

# 小姜锅贴

地址： 十全街1019号

鲜肉、招牌牛肉和虾仁 、牛肉粉丝汤、牛杂粉丝汤

# 定慧寺巷

苏东坡曾来这打卡

# 紫笑香

夏日“快乐水”

地址： 定慧寺巷53号

苏式绿豆汤

# 兴奋羊肉老店

地址： 定慧寺巷83号

羊肉火锅、白切羊肉、羊杂、羊血、羊糕、红烧羊肉

# 王氏林记烧饼

地址： 定慧寺巷5号

早餐铁三角：烧饼、油条、豆浆

# 陆家市食品店

地址： 定慧寺巷26号

常熟特产：蜜饯、杨梅干、话梅、猫耳朵

# 双塔市集

逛完定慧寺巷，过吴王桥，苏州网红市集，在定慧寺巷东侧

# 喜有福

原名"老客满煎饼"，开店二十多年

葱油鸡蛋饼、韭菜鸡蛋饼、豆腐花

# 老意和

馄饨花样很多、汤圆、鲜牛肉锅贴

# 裕永盛

生煎

# 棠芝坊

非遗代表性传承人陈巧文

梅花糕、海棠糕

# 小热荤

杂粮煎饼(非苏州本地美食)

# 大孃孃

赤豆糊糖粥、赤豆糊圆子、酒酿小圆子、南塘鸡头米

# 面见阳春

焖肉面、大排面、虾仁面、苏式冷面和凉皮(夏季)

# 书院巷

有书香、更有美食香

# 老王蛋饼

老店20多年

地址： 书院巷金狮河沿1号

招牌油酥蛋饼(韭菜、葱花、包菜)

# 明月楼糕团

地址： 书院巷86号

猪油糕、松糕、拉糕、炒肉馅团子(夏)

# 潘记隆烧饼

地址： 东大街498号

“面衣饼“、油氽、鸡蛋

# 杜三珍卤菜

中华老字号

地址： 书院巷69号(分店：定慧寺巷23号)

五香牛肉、酱鸭、猪头肉、酱肉、五香小肉、熏肠、熏肚

# 木缘新豆花

地址： 书院巷61号

冰镇红豆豆花口感更新广式甜品双皮奶

# 黎里王记辣脚

地址： 书院巷123号

黎里辣脚

# 矮脚楼馄饨

地址： 书院巷109号

三鲜馄饨、生煎馒头、荠菜馄饨、虾仁馄饨

# 观前街

苏州第一商业街

# 姑苏区老黄苏式蛋饼

开业十多年

地址： 建新巷8号(和基广场北门地铁3号口)

蛋饼

# 长发月饼

地址： 大儒巷与临顿路交叉路口往东约50米

鲜肉月饼

# 哑巴生煎

地址： 临顿路温家岸12号

生煎

# 蛮好阁包子馆

地址： 旧学前9号

招牌净素菜包子

# 陆稿荐

中华老字号

地址： 临顿路与大儒巷交叉路口西南侧

熟食店：酱汁肉、五香酱肉、五香牛肉、酱猪头肉

# 采芝斋

中华老字号，非遗苏式糖果

地址： 观前街91号

松仁粽子糖、芝麻薄皮、蛋黄花生、轻松糖、轻松胡桃

# 叶受和

中华老字号

地址： 观前街75号

苏式糕点：小方糕、四色片糕、八珍糕、虾籽鲞(xiǎng)鱼

# 稻花香

中华老字号

地址： 观前街72号

枣花酥、山楂锅盔、牛舌饼

# 黄天源

中华老字号，糕团大王，舌尖上的中国

地址： 观前街碧风坊与观巷交叉路口西北侧

薄荷糕、玫瑰糕、猪油糕、撑腰糕、青团子、定胜糕、重阳糕

# 松鹤面馆

松鹤楼创办

地址： 观前街太监弄72号(近玄妙观)

姑苏卤鸭面

# 近水台

中华老字号面馆

地址： 观前街太监弄127号

苏式面：焖肉面、爆鱼面、爆鳝面、蹄膀面，苏轼糕团

# 同得兴

地址： 嘉馀(yú)坊6号

枫镇大肉面(5到10月供应、舌尖上的中国)

# 聚新春

地址： 嘉馀坊21号

焖肉面、爆鱼面、糟鸡面、油氽紧酵馒头("兴隆馒头")

# 皮市街

苏州最甜蜜的一条街

# 潘玉麟糖粥

地址： 史家巷46号

潘玉麟糖粥：赤豆糊圆子

# 杨招娣糕点

开业二十多年

地址： 皮市街277号

只卖赤豆糕、赤豆猪油糕、枣子豆沙糕

# 赤小豆饮食店

地址： 皮市街325号

苏式绿豆汤、秘制鸡脚、赤豆糊小圆子、酒酿小圆子、春卷、炸鸡腿

# 十年麻辣烫

开业二十多年

地址： 皮市街296号

麻辣烫

# 宗霞鸡蛋饼

开业二十多年

地址： 皮市街296号

油酥蛋饼+肉串、火腿肠、培根

# 吴趋坊

千年古坊，繁华依旧

# 乐惠馄饨

地址： 吴趋坊44号

泡泡馄饨(肉馅只有一点点)、拌馄饨、汤圆

# 天桥炒面

地址： 吴趋坊72号

炒面、卤香干、鸡腿

# 汤香源面馆

1996年开始营业，原名汤家巷面馆

地址： 吴趋坊84号

传统江汤面+大排、焖肉、鳝糊、爆鱼、狮子头

# 陈记泡泡馄饨

地址： 吴趋坊108号

泡泡馄饨、芝麻汤圆、肉汤圆

# 老城隍庙锅贴

地址： 吴趋坊112号

锅贴(非苏州传统小吃)+胡辣汤

# 西北街

北寺塔下，苏式人家

# 山塘老街

地址： 西北街15号

灌汤小笼包、荠菜肉馄饨、泡泡小馄饨、苏式面、桂花鸡头米

# 吴盛兴

地址： 西北街295号

绉(zhòu)纱汤包、生煎、泡泡馄饨、苏式面、绿豆汤

# 西北街豆浆店

开业15年

地址： 西北街100号

豆浆、豆腐花、油条、粢饭糕、荷叶包美人、绿豆百合汤、喜蛋

# 美味斋

地址： 西北街111号

爆鱼面、凉面

# 剑雄炒货蜜饯店

地址： 西北街147号

话梅、花生、山楂

# 吴记小园楼

地址： 西北街151号

松鼠鳜鱼、响油鳝糊、清炒虾仁、莼菜银鱼羹、葱油拌面

# 原怡园面馆

地址： 西北街207号

枫镇大肉面、虾籽白斩鸡(限量)

# 西中市

金闾门内商市兴

# 三德利绉纱汤包馆

地址： 西中市21号

绉纱汤包

# 老陆稿荐

中华老字号

地址： 西中市39号附近

五香小肉、五香酱肉、焖肉、苏式面

# 朱鸿兴

创建于20世纪30年代，老面馆

地址： 西中市48号(分店：宫巷108号)

枫镇大肉面、三虾面、酱鸭面、冻鸡面

# 陈记一品斋粽子

地址： 西中市56号(总店位于司前街)

大肉粽、蛋黄肉粽、净素粽、细豆沙粽

# 闾门姚记豆浆

30多年，24小时营业

地址： 西中市60号

招牌咸豆浆(虾米、油渣、油条碎)、豆腐花、油条、粢饭糕、荷叶包美人

# 鼎盛鲜

地址： 西中市84号

几近失传的美食：炖粉烫面饺、三鲜千张包

# 五德居

地址： 西中市135号

白斩鸡(夏)、狮子头(冬)

# 老陆家汤包馆

地址： 西中市146号

苏式汤包

# 山塘街

姑苏第一名街

# 马栋佩烧卖

二十多年

地址： 广济路新民桥菜市场内

烧卖个头大

# 义昌福

百年老店

地址： 广济路新民桥菜市场内

鲜肉包、三丁包、青菜香菇包、刀切馒头、香葱花卷

# 江南蟹壳皇

地址： 广济路新民桥菜市场内

芝麻糖蟹壳黄(甜)、葱油蟹壳黄(咸)、萝卜丝蟹壳黄

# 东方卤菜店

# 松鹤楼

松鼠鳜鱼、清炒虾仁、响油鳝糊

松糕、猪油糕、重阳糕
