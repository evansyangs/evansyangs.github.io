# 5. 无序列表

可以使用*，+，-表示无序列表

注意： 符号后面需有空格,无序列表需与上方文字保持一行空格才有效 如： （空行）

  • 序号

  • 序号

# 6. 有序列表

使用数字和点即可

注意： 符号后面也需有空格，此处有无空行都一样，与无序列表注意区分 如：

  1. 序号1
  2. 序号2
  3. 序号3

# 7. 文字引用

使用符号>来表示此段文字为引用文字 如：

苟利国家生死以，岂因祸福避趋之
