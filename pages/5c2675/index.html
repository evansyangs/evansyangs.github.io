1. SSR教程

# 1.1 简介 (opens new window)

Shadowsocks（简称SS）是一种基于Socks5代理方式的加密传输协议，也可以指实现这个协议的各种开发包。当前包使用Python、C、C++、C#、Go语言、Rust等编程语言开发，大部分主要实现（iOS平台的除外）采用Apache许可证、GPL、MIT许可证等多种自由软件许可协议开放源代码。Shadowsocks分为服务器端和客户端，在使用之前，需要先将服务器端程序部署到服务器上面，然后通过客户端连接并创建本地代理。

在中国大陆，本工具广泛用于突破防火长城（GFW），以浏览被封锁、遮蔽或干扰的内容。2015年8月22日，Shadowsocks原作者Clowwindy称受到了中国政府的压力，宣布停止维护此计划（项目）并移除其个人页面所存储的源代码。

为了避免关键词过滤，网民会根据谐音将ShadowsocksR称为“酸酸乳”[注 1]（SSR），将Shadowsocks称为“酸酸”（SS）。另外Shadowsocks的标志为纸飞机，而专门提供Shadowsocks或类似软件（如V2ray）服务器的网站也被称为“机场”。

下面介绍其在各个平台上的使用方法，请各位慢慢食用。

# 1.2 Windows

# 1.2.1 软件下载

点击下载客户端 (opens new window)

# 1.2.2 导入节点

  • 解压客户端，双击shadowsocksr4.0的客户端（打不开的话用2.0的）

  • 方法一：

    1. 复制你的ssr://链接或打开你的二维码
    2. 找到系统托盘菜单中的小飞机图标右键
    3. 点击剪贴板批量导入ssr://链接或二维码扫描
    1. 将系统代理模式设置为PAC模式
    2. 将代理规则设置为绕过局域网和大陆

  • 方法二：

    1. 复制你的https://订阅链接
    2. 找到系统托盘菜单中的小飞机图标右键
    3. 打开SSR服务器订阅设置
    1. 点击Add添加一个订阅，将复制的链接填入右侧网址框内，点击确定
    1. 找到系统托盘菜单中的小飞机图标右键
    2. 点击更新SSR服务器订阅（不通过代理）

# 1.2.3 选择节点

  1. 找到系统托盘菜单中的小飞机图标右键
  2. 点击服务器，选择一个群组，最后选择一个节点单击
  1. 打开浏览器，尝试访问www.google.com (opens new window)

# 1.3 IOS

# 1.3.1 软件下载

请联系我获取美区账号与密码

# 1.3.2 导入节点

  • 方法一：
    1. 复制你的ssr://链接将其转换为二维码（点击本链接 (opens new window)）或打开你的二维码
    2. 点击软件左上角扫描二维码
    3. 将全局路由设置为配置
  • 方法二：
    1. 复制你的https://订阅链接
    2. 点击软件右上角+号
    3. 类型中选择Subscribe
    4. 将你的订阅链接填入URL中，点击完成

# 1.3.3 选择节点

  1. 点击首页上任一服务器节点即可
  2. 打开浏览器，尝试访问www.google.com (opens new window)

# 1.4 Android

# 1.4.1 软件下载

点击下载APP (opens new window)

# 1.4.2 导入节点

  • 方法一：
    1. 复制你的ssr://链接或打开你的二维码
    2. 点击软件左上角ShadowsocksR文字
    3. 点击右下角+号，从菜单中选择从剪贴板导入或扫描二维码
  • 方法二：
    1. 复制你的https://订阅链接
    2. 点击软件左上角ShadowsocksR文字
    3. 点击右下角+号，从菜单中选择添加/升级SSR订阅
    4. 在弹出的界面中点击添加订阅地址
    5. 将你的订阅链接填入其中，点击确定
    6. 打开自动更新后，点击确定并升级

# 1.4.3 选择节点

  1. 点击软件左上角ShadowsocksR文字
  2. 选择其中的任一节点后，在下方功能设置中，将路由设置为绕过局域网及中国大陆地址
  3. 点击右上角的小飞机进行连接
  4. 打开浏览器，尝试访问www.google.com (opens new window)

# 1.5 MacOS

# 1.5.1 软件下载

点击下载客户端 (opens new window)

# 1.5.2 导入节点

  • 方法一：
    1. 复制你的ssr://链接将其转换为二维码（点击本链接 (opens new window)）或打开你的二维码
    2. 点击服务器，选择扫描屏幕上的二维码
  • 方法二：
    1. 复制你的https://订阅链接
    2. 点击服务器，选择编辑订阅
    3. 点击左下角+号，在订阅地址中填入你的订阅链接

# 1.5.3 选择节点

  1. 点击小飞机
  2. 在服务器中选择任一节点后，打开绕过局域网及中国大陆地址
  3. 点击右上角的小飞机进行连接
  4. 打开浏览器，尝试访问www.google.com (opens new window)
