    # 1.1 Python的历史

    大多数的文章都会以Python的历史为开头，作为一名菜鸟（YYF牛逼！）作者，这篇文章自然也就稍微接地气些，跟大家简简单单唠叨个几句。(其实我也不太知道这段历史，看别人文章的时候一般会选择跳过这节内容#滑稽)

    # 1.2 与MATLAB的不同

    MATLAB是什么？（这个问题...俗话说的好，MATLAB除了生孩子，什么都能干！）

    言归正传，作为一名数学🐶，MATLAB就是我本科四年吃饭的家伙，只要一提到科学计算——掏出电脑，掀开盖子，双击MATLAB图标，（泡上一碗香喷喷的老坛酸菜牛肉面，等你美美地解决完今晚的宵夜之后，）终于！MATLAB它弹出了那熟悉而又久违的主界面！

    其实我觉得Python和MATLAB区别主要如下：

    • MATLAB贵！贼🐔儿贵！（不过，福建大三本的师生免费...小声bb，想必是MathWorks的扶贫计划吧）

    • MATLAB大！MATLAB R2019在线安装个主程序要1个G...安装Toolbox的时候还经常莫名断开连接...（你说...你对得起5元10G的seu网费吗！）

    • MATLAB真香！无脑doc，自从接触了Python（挖坑！挖坑！无底的坑啊）...我人没了...

    以上，只是我个人的主观感受啦！（欢迎在下方留言区交流！）

    # 2. Python相关软件安装

    工欲善其事必先利其器 —— 孔子（春秋）《论语·卫灵公》

    这边我只简单介绍Win10系统的安装（其它系统的请自行百度/谷歌啦！）

    • Step1：打开这个网站ANACONDA (opens new window)

    • Step2：单击网页右上角深绿色且够醒目的Download (opens new window)

    • Step3：向下滚动网页，直至看到Python 3.7 version这几个大字！

    • Step4：单击大字下方深绿色且够醒目的Download (opens new window)

    • Step5：别问！问就是下载它！干他！

    • Step6：默默地从身后掏出一袋Lipton香浓原味奶茶粉，剪开其一个角，将粉末倒入精致的陶瓷杯中，倒入适量的热开水，用一把闪闪发光的勺子缓慢地顺时针搅拌，注意不要让勺子碰到杯壁发出“叮~叮~叮~”的奇怪声响！当你优雅地喝完一杯82年的奶茶后，想必应该也下载好了吧~

    • Step7：双击打开安装包，下一步...下一步...下一步...下一步...

    • Step8：安装完毕！

    👇接下来让我们看看别人家写的正经的安装教程吧！👇
