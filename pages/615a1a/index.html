# 2.2 经验二

【经验分享】软件工程/金融工程保研推免夏令营 (opens new window)

同济大学软件学院的夏令营时间非常短，不算报道的话只有一天半，而且放在周末，所以跟其他学校夏令营时间一般都不会冲突，而且紧接着就是上财的夏令营也很方便，所以我就参加了这个夏令营感受学术氛围（并且拿个保底（逃….因为时间很短，所以基本全部都在考核。第一天上午是导师方向介绍，下午笔试，晚上一个讲座，第二天早上面试，下午就各回各家。 笔试分为专业笔试和英语笔试，面试分为专业面试和英语面试。

  • 专业笔试两个小时，考试内容是：
    • 一道数学分析的导数证明题
    • 一道最短路径搜索的算法伪代码（Dijkstra’s algorithm）
    • 一个MOBA游戏的面向对象设计（OOD），用UML来表示
    • 一个MOBA游戏的数据库设计
    • 自动检测停车位的人工智能模型训练流程（计算机视觉方向）
  • 英语笔试35分钟，考试内容是：
    • 十个英文缩略语写全称，内容很宽泛，看个人平时积累为主
    • 两道英译中，第一道是两个最短路径搜索算法的差异比较，应该是英文教材上的摘录，第二道感觉是通讯信号领域的论文Abstract，讲的是视频流传输
    • 一道中译英，材料是关于软件开发流程的
  • 专业面试因人而异，有些人的面试比较challenging，有些就很水，我碰到的就很水，问我本科的学校挺好的为什么不留在本校啊，报了哪些夏令营啊，要是给你offer了你会不会来啊之类的
  • 英语面试会让你自我介绍，然后根据你的自我介绍提一两个问题就pass了，也比较轻松

同济大学软件学院的夏令营入营100人，来了90多个人，给了50个左右的优秀营员，住的酒店很不错，吃的不是特别好，但整体流程比较轻松，还是值得一去的。

# 2.3 经验三

同济大学软件学院专硕复试过程及2014年回忆版的题目 (opens new window)

2014年软院专硕第一志愿复试题回忆版（去年一志愿和调剂分开复试的）

一、填空题

  1. 操作系统死锁的原因是____________________________________。
  2. 算法时间复杂度的意思是__________________________________。
  3. 数据库中DDL是指_______，DML是指________________。
  4. CPU由_____________和________________两部分组成的。
  5. 二叉树的第六层最多有________个节点。
  6. 关系数据库的结构___________________。（这道题就是这样，不懂什么意思）
  7. 图片在计算机中是以点阵的形式存储的，M*N点阵，存储每一个点叫做_______.

二、简答题

  1. 试比较出栈和队列的区别
  2. 瀑布模型是软件工程中常用的开发模型，画出该瀑布模型
  3. 写出声音信号到数字信号的三个步骤
  4. 写出下面SQL语句所实现的功能（下面给出了两个很简单的SQL语句）
  5. 计算机病毒的传播方式，传播途径有哪些

三、分析题

  1. 操作系统的功能有哪些，列举出你所知道的操作系统（至少3种）
  2. 说出TCP/IP的层次以及每个层次的功能
  3. 数据库设计题 老师（属性：编号 姓名 所教课程ID 课程名称....） 学生（属性：学号 课程ID 授课教师名称....） 课程（属性：课程ID 任课教师....） 专业（属性：专业ID 包含课程.....） 画出E-R图

（有一道选择题没回忆上来，还有分析题最后一道对象属性不是很准确，简答题SQL语句不记得了，不过那个SQL语句很简单很简单，这基本上是所有的题了）

# 2.4 经验四

2019年双非跨考上岸同济大学软件学院考研复试经验分享，笔试真题回忆！ (opens new window)

一、笔试部分

第一题

矩阵A为m×n阶 m>n r(A)＝n 证明 ATAA^T AA​T​​A可逆 （15分）

第二题

（1）给了几个城市和各城市之间距离，求其中两个城市的最短路径（5分）
（2）迪杰斯特拉或弗洛伊德算法伪代码（15分）

第三题

（1）画哈夫曼树（2）根据第一问写出相应哈夫曼编码（30分）

第四题

自动停车的问题，让你尽可能详细的描述出如何去实现（30分）

英翻中

（1）文件系统 （2）最小生成树 不难 但感觉有点长 （每个15分）

中翻英

机器学习（跟机器学习的维基百科上的最前面几段差不多）（20分）

二、英语面试

只有一个老师，面对面坐着，不用读文章，先自我介绍，跨考的会让你介绍本科专业，问到家乡，会问到毕设，会问到你选的三个导师以及为什么选他们，英文面试比较短，大概五六分钟的样子

三、专业面试

十到十五个导师的样子，你坐在长桌的一头，老师围着长桌坐着。 一进去先自我介绍，然后本科就是计算机的问的专业知识比较多，跨专业的会问你本科专业情况，问了毕设， 对什么方向感兴趣 选了哪三个导师，为什么选他们，本科成绩在年级中处于什么位置 老师比较注重项目和编程能力 对于社团经历不感兴趣，我是我们学校电脑协会会长，但是老师一句都没提协会的事，一直在问我项目。 每个人15分钟是有计时的，最后看时间到了就让我出来了，临走之前说我专业学的课程和他们比较远，问我有没有做好准备。最后我都起身要出去了，又问我本科是不是211
